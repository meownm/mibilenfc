# Mobile (Android, Java) — Demo Client

Это минимальный проект Android (Java) для демо‑сценария:

1. Сфотографировать паспорт.
2. Отправить фото на backend `/api/passport/recognize`.
3. Показать распознанные параметры.
4. По кнопке перейти к NFC и считать eMRTD (BAC) используя параметры (номер документа/дата рождения/дата окончания).
5. Вынуть фото лица из DG2 и отправить вместе с параметрами в backend `/api/passport/nfc`.

Важно: проект ориентирован на устройство с NFC. Для чтения eMRTD используется библиотека `jmrtd`.
В демо включены файлы Gradle и каркас кода. Для запуска проверьте зависимости в Android Studio и права.

Файл настройки backend: `app/src/main/java/com/demo/passport/BackendConfig.java`.
